# ぎゅ~れっと (GyuLette)

楽しいルーレット抽選Webアプリケーション

## 特徴

- 🎯 視覚的なルーレットで抽選を実行
- ⭐ 当選確率アップ設定
- 🎨 参加者ごとのカラフルな表示
- 💾 LocalStorageによる状態の自動保存
- 📱 レスポンシブデザイン

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# 静的エクスポート
npm run export
```

## 技術スタック

- Next.js 14
- React 18
- TypeScript 5
- Tailwind CSS 3

## 使い方

1. 参加者を追加
2. 除外設定や⭐設定を調整
3. 「スピン!」ボタンをクリックして抽選実行
4. 当選者が表示されます
